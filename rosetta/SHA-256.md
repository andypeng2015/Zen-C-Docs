# SHA-256

```zc
import "std/crypto/sha256.zc"

fn main() {
    let hash = Sha256::hash("Rosetta code");
    println "{hash}";
}
```

**Output:**

```
764faf5c61ac315f1497f9dfa542713965b785e5cc2f707d6468d7d1124cdfcf
```
